# OpenGameArt reference sources

This pack was generated to match the requested manifest and uses these OpenGameArt CC0 pages as style/category references.

Important: the generated WAV files in this ZIP are procedural/foley-style approximations created in this session. The OpenGameArt ZIP/7Z binaries were not embedded directly because this runtime could not import external binary archives. If you manually download the source packs, they can be remapped/reprocessed later into the same manifest paths.

## Unity import suggestion

Import ZIP so the top-level `Assets/_Project/Audio` folder lands inside the Unity project. Keep these as decompressed WAV or set compression to ADPCM/Vorbis depending on your platform target.

Suggested:
- Short SFX: Decompress On Load, mono, preload on.
- Campfire/forest ambience: Streaming or Compressed In Memory, loop enabled.
