# Apex Shift Audio Assets v3 realistic-ish

Generated 66 WAV assets matching the supplied manifest.

- Sample rate: 48,000 Hz
- Format: 16-bit PCM WAV
- SFX channels: mono
- Ambience loops: stereo
- Unity path: `Assets/_Project/Audio`

## What changed vs previous attempts

This version separates sound families more strongly:

- spear swing: airy directional whoosh + shaft vibration
- spear/arrow flesh hit: dull thud + wet/cloth transient
- bow release: plucked string/twang + small wood snap
- arrow fly: subtle high pass-by
- wood chop: low wooden thock + splinter crack
- stone hit: bright ping + stone tick
- trap snap: sharp spring/wood clamp
- creature sounds: different pitch ranges and envelopes for Varnak, Grazer and small prey
- footsteps: separate grass/dirt/stone textures
- ambience: stereo campfire and forest loops

See `audio_manifest.json` for IDs and Unity mappings. See `SOURCES.md` for OpenGameArt reference pages.
